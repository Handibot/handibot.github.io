Heads-Up on MACROS

Macros are a bit of a work in progress. We have been updating them to remove a few bugs and to add more functionality. You'll want to make sure you have the latest installed on your tool.

To install the latest set of Macros, download the the Macro Install App for your tool (e.g. V2 or V1update) from here (at docs.handibot.com), then add it as an App from the home screen. When you run the Install App you'll have the option of installing all the Macros or just individual Macros. You can leave the App on your tool to re-install Apps again, or you can delete it. You can always get the most up-to-date Macros and Apps from the docs page.

Let us know if you have any problems.
