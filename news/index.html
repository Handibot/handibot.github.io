The NEWS Is that there is a Major New Software Update for FabMo (Ver.1.6.0)

The update includes significant improvements and fixes to the FabMo software.

In the future, updates will happen straightforwardly using the Configuration Manager of the FabMo dashboard. But for this release, you will need to carry out the update from the Updater Window.

Instructions for carrying out the update using two different methods are available here. Most users will want to use the first method, the Internet Update. But, if your Handibot has poor or non-existent access to the internet, you can use the second method that does the update with only an AP connection to your device (assuming you can download the update file from the internet to your device).

Each link includes full instructions.
